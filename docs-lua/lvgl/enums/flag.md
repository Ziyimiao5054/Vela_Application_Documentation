# FLAG

| 常量 | 值 | 说明 |
|------|----|------|
| **HIDDEN** | 1 | 隐藏对象 |
| **CLICKABLE** | 2 | 对象可被点击 |
| **CLICK_FOCUSABLE** | 4 | 点击时可获得焦点 |
| **CHECKABLE** | 8 | 可切换选中状态 |
| **SCROLLABLE** | 16 | 内容可滚动 |
| **SCROLL_ELASTIC** | 32 | 滚动回弹效果 |
| **SCROLL_MOMENTUM** | 64 | 滚动惯性 |
| **SCROLL_ONE** | 128 | 一次只滚动一个方向 |
| **SCROLL_CHAIN_HOR** | 256 | 横向滚动链 |
| **SCROLL_CHAIN_VER** | 512 | 纵向滚动链 |
| **SCROLL_CHAIN** | 768 | 滚动链（SCROLL_CHAIN_HOR \| SCROLL_CHAIN_VER） |
| **SCROLL_ON_FOCUS** | 1024 | 获焦时自动滚动至可见 |
| **SCROLL_WITH_ARROW** | 2048 | 方向键滚动 |
| **SNAPPABLE** | 4096 | 对齐吸附 |
| **PRESS_LOCK** | 8192 | 按下锁定 |
| **EVENT_BUBBLE** | 16384 | 事件冒泡 |
| **GESTURE_BUBBLE** | 32768 | 手势冒泡 |
| **ADV_HITTEST** | 65536 | 高级命中测试 |
| **IGNORE_LAYOUT** | 131072 | 忽略父容器布局 |
| **FLOATING** | 262144 | 浮动于布局之外 |
| **OVERFLOW_VISIBLE** | 1048576 | 超出边界仍可显示 |
| **LAYOUT_1** | 8388608 | 自定义布局标志 1 |
| **LAYOUT_2** | 16777216 | 自定义布局标志 2 |
| **WIDGET_1** | 33554432 | 自定义控件标志 1 |
| **WIDGET_2** | 67108864 | 自定义控件标志 2 |
| **USER_1** | 134217728 | 用户自定义标志 1 |
| **USER_2** | 268435456 | 用户自定义标志 2 |
| **USER_3** | 536870912 | 用户自定义标志 3 |
| **USER_4** | 1073741824 | 用户自定义标志 4 |
