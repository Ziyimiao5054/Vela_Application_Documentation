# SCR_LOAD_ANIM

| 常量 | 值 | 说明 |
|------|----|------|
| **NONE** | 0 | 无动画，直接切换 |
| **OVER_LEFT** | 1 | 新屏幕从左侧覆盖 |
| **OVER_RIGHT** | 2 | 新屏幕从右侧覆盖 |
| **OVER_TOP** | 3 | 新屏幕从上方覆盖 |
| **OVER_BOTTOM** | 4 | 新屏幕从下方覆盖 |
| **MOVE_LEFT** | 5 | 旧屏幕左移，新屏幕右侧进入 |
| **MOVE_RIGHT** | 6 | 旧屏幕右移，新屏幕左侧进入 |
| **MOVE_TOP** | 7 | 旧屏幕上移，新屏幕下方进入 |
| **MOVE_BOTTOM** | 8 | 旧屏幕下移，新屏幕上方进入 |
| **FADE_ON** / **FADE_IN** | 9 | 渐入动画 |
| **FADE_OUT** | 10 | 渐出动画 |
| **OUT_LEFT** | 11 | 旧屏幕向左滑出 |
| **OUT_RIGHT** | 12 | 旧屏幕向右滑出 |
| **OUT_TOP** | 13 | 旧屏幕向上滑出 |
| **OUT_BOTTOM** | 14 | 旧屏幕向下滑出 |
